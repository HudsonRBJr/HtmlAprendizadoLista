Input Attributes

possui como principais atributos type, que recebe como valor o tipo do input. A tag em um formulário HTML é um elemento usado para criar campos de entrada interativos, nos quais as pessoas usuárias podem inserir dados.

Exemplo:

< form action="/action_page.php">
< label for="fname">First name:< /label>
< input type="text" id="fname" name="fname" value="John">
< label for="lname">Last name:< /label>
< input type="text" id="lname" name="lname" value="Doe">

< input type="submit" value="Submit">
< /form>
