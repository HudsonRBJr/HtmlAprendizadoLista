Html Entities

HTML Entities - HTML symbols - HTML characters: São aqueles símbolos não interpretados pelo navegador. Você não precisou usar aqueles caracteres especiais que não estão presentes no seu teclado. Pois o navegador ou seu Editor de texto (Sublime, Notepad++ e etc...) não consegue interpretar muito bem.
