HTML Heading

O elemento HTML HEADINGS representa um grupo de suporte introdutório ou navegacional. Pode conter alguns elementos de cabeçalho mas também outros elementos como um logo, seções de cabeçalho, formulário de pesquisa, e outros.

Tais headings seriam:< h1 >, < h2 >, < h3 >, < h4 >, < h5 >, < h6 >, Esses sao todos os headings que podem ser usados.

Exemplo HTML:

< h1>A< /h1>
< h2>B< /h2>
< h3>C< /h3>
< h4>D< /h4>
< h5>E< /h5>
< h6>F< /h6>
