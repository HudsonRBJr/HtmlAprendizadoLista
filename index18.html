Html Classes

As classes são uma forma de identificar um grupo de elementos. Através delas, pode-se atribuir formatação a VÁRIOS elementos de uma vez.

Exemplo Html:

Aviso!!!


pra total customizaçao de uma div é necessario usar a tag style entre a tag < head>< /head> ou fazer um arquivo proprio tipo "style.css"

exemplo de como fazer no html sem criar um arquivo .css: < style>
.city {
background-color: tomato;
color: white;
border: 2px solid black;
margin: 20px;
padding: 20px;
}
< /style>.


Agora Dentro do body:


< div class="city">
< h2>London< /h2>
< p>London is the capital of England.< /p>
< /div>
