HTML Formatting

O HTML Formatting permite que os desenvolvedores estruturem o conteúdo de uma página da web de forma clara e organizada.

Exemplo:

< b> - Bold text
< strong> - Important text
< i> - Italic text
< em> - Emphasized text
< mark> - Marked text
< small> - Smaller text
< del> - Deleted text
< ins> - Inserted text
< sub> - Subscript text
< sup> - Superscript text
