Oque Seria HTML attributes?

Atributos HTML (em inglês html attributes") são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Os atributos HTML são um modificador de um tipo de elemento HTML.

de atributos temos as tags: A para links, IMG para colocar imagems e finalmente STYLES para adicionar stylos a elementos, etc...

Exemplos:

< a href="https://www.w3schools.com">Visite O Site Sobre HTML < /a >
< img src="Sua mae.jpg" width="500" height="600" >
< p style="color:red;">Este é um paragrafo vermelho.< /p >

ficariam assim:

Visite o Site Sobre HTML

IMG aqui deveria ficar uma imagem
