Hmtl Iframes

O iFrame é uma tag usada para inserir uma página HTML em outra. O uso dessa tag é fundamental por dois motivos principais. Primeiro, porque permite que você insira a página enquanto mantém a estrutura.

Exemplo em Html:

< h2>HTML Iframes< /h2>
< p>You can use the height and width attributes to specify the size of the iframe:< /p>
< iframe src="demo_iframe.htm" height="200" width="300" title="Iframe Example">< /iframe>
